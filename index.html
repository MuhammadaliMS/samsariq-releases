MCP

Built for AI agents.

Samsariq exposes a full Model Context Protocol interface. Let your AI agents control everything.

Start/stop recordings via MCP
Query transcripts, notes & summaries
Schedule recurring sessions
Works with Claude, GPT & more
mcp_config.json
// Add to your MCP client config
{
  "mcpServers": {
    "samsariq": {
      "command": "/path/to/samsariq",
      "args": ["--mcp"]
    }
  }
}
// Then ask your AI agent:
// "Start recording my standup"
// "Summarize today's meetings"
