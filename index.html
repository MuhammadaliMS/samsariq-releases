Features

Everything you need. Nothing you don't.

Samsariq is designed to do one thing exceptionally well: capture your meetings and turn them into actionable intelligence, all without leaving your machine.

Real-Time Transcription

Powered by Whisper.cpp running locally with GPU acceleration. Get instant, accurate transcripts as your meeting unfolds.

AI Summaries

Generate comprehensive meeting summaries, action items, and key decisions using local LLMs via Ollama or cloud APIs.

Privacy First

Your audio and transcripts never leave your device. No cloud processing, no third-party servers. Your conversations stay yours.

MCP Integration

Built-in Model Context Protocol support lets AI agents control recordings, query transcripts, and manage schedules programmatically.

Speaker Diarization

Automatically identifies and labels different speakers in your meetings, making it easy to follow who said what.

Cross-Platform

Native desktop app for macOS and Windows built with Tauri. Fast startup, minimal memory usage, and deep OS integration.

How It Works

Three steps to clarity

No setup hassle. No accounts required. Just install, record, and get your meeting notes.

1

Start Recording

Hit record or use the global push-to-talk shortcut. Samsariq captures both your microphone and system audio simultaneously.

2

Watch It Transcribe

Whisper.cpp processes your audio locally with GPU acceleration. See real-time transcripts appear as the conversation flows.

3

Get AI Summaries

When you're done, get instant AI-generated summaries with action items, decisions, and key takeaways via local or cloud LLMs.

MCP Integration

Built for AI agents

Samsariq exposes a full Model Context Protocol interface. Let your AI agents control recordings, query transcripts, and schedule meetings.

Agent-First Architecture

The MCP server runs over a Unix domain socket with a stdio companion binary. Your AI agents can start recordings, fetch transcripts, and manage meeting schedules — all programmatically.

✓ Start/stop recordings via MCP
✓ Query transcripts and summaries
✓ Schedule recurring meetings
✓ Works with Claude, GPT, and more
mcp_config.json
// Add to your MCP client config
{
  "mcpServers": {
    "samsariq": {
      "command": "samsariq_mcp",
      "transport": "stdio"
    }
  }
}
// Then ask your AI agent:
// "Start recording my standup"
// "Summarize today's meetings"
// "Schedule weekly sync at 10am"
