MCP Integration

Built for AI agents

Samsariq exposes a full Model Context Protocol interface. Let your AI agents control recordings, query transcripts and notes, and schedule sessions.

Agent-First Architecture

The MCP server runs over a Unix domain socket with a stdio companion binary. Your AI agents can start recordings, fetch transcripts and notes, and manage schedules — all programmatically.

✓ Start/stop recordings via MCP
✓ Query transcripts, notes, and summaries
✓ Schedule recurring sessions
✓ Works with Claude, GPT, and more
mcp_config.json
// Add to your MCP client config
{
  "mcpServers": {
    "samsariq": {
      "command": "/path/to/samsariq",
      "args": ["--mcp"]
    }
  }
}
// Then ask your AI agent:
// "Start recording my standup"
// "Summarize today's meetings"
// "Schedule weekly sync at 10am"
